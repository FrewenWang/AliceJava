### 抽象工厂模式介绍

抽象工厂模式也是创建者模式之一。

### 抽象工厂模式定义

为创建一组相关或者相互依赖的对象提供一个接口。而不需要指定他们的具体类

分析：很简单。上一章我们介绍简单工厂模式，都需要指定产品的具体类。哪怕是在具体的工厂方法里面来实例化具体的类。
还是通过反射类实例化具体的产品类。都是需要执行产品的具体类。

而抽象工厂模式是创建一组相关或者相互依赖的对象提供一个接口。而不需要指定他们的具体类
